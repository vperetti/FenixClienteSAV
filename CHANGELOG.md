# Changelog

## v3.0 - PuTTY 0.74 (Janeiro 2021)
- Migração completa para PuTTY 0.74 com suporte SSH2 moderno
- Reescrita das modificações MEXIDAMINHA para nova arquitetura do PuTTY
- Adição de `printer_abort_job()` para melhor controle de impressão
- Function pointer `ShellExecuteA` na vtable do TermWin
- Compilação com MSVC (Visual Studio)
- Instaladores Inno Setup por cliente (2021a)

## v2.0 - PuTTY 0.56 Evoluído (2008-2011)
- Evolução das modificações MEXIDAMINHA
- Menus personalizados: Relatórios, Ajuda, Suporte, Portal Intranet, Terminal Seguro
- Interceptação de URLs via LP ANSI (ESC[5i/ESC[4i)
- Execução de fenixReport.jar via prefixo "httpjr"
- Interface traduzida para português
- Múltiplos instaladores por cliente (Britamaq, Tecnoeste, FW, EBR, Romac, etc.)

## v1.0 - PuTTY 0.56 Inicial (2004-2005)
- Primeiro fork do PuTTY para uso com ERP SAV
- Modificações iniciais MEXIDAMINHA
- Renomeação do executável para ClienteSAV.exe
